Objective

In order to reduce overall page weight, test to see if we can use smaller image dimensions without a big hit to quality, in order to reduce file size, which in turn would result in faster page load times.

Conclusion

Reducing image dimensions is effective in reducing file size of images without suffering much lack of quality at all. This will result in both page weight and page load reduction on any page across the site with hero images.
